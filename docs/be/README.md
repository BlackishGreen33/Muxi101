### 主要技术栈

团队主要采用 Go 和 Python 作为开发语言。此外配套使用了 Gin/Gorm/gRPC/Flask/Gunicorn/SQLAlchemy。数据库和中间件常用的是 Redis/MySQL/MongoDB/InfluxDB/RabbitMQ。异常监控使用 Sentry。容器和编排使用 Docker/K8s/K3s 等等。IAAS 层使用阿里云。
